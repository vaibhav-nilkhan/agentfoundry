## Installation

All skills available via **one-click install**:

```bash

agentfoundry install <skill-name>

# Examples
agentfoundry install cross-platform-tool-adapter  # Production
agentfoundry install github-pr-analyzer            # Beta
agentfoundry install viral-content-predictor --tier experimental  # Experimental
```

## Validation Methodology

Each skill was validated using a **rigorous 6-point checklist**:

1. ✅ **Real Problem**: Can we find GitHub Issues/Discussions?
2. ✅ **Recent**: Issues from 2025?
3. ✅ **Multiple Developers**: More than 1-2 people affected?
4. ✅ **Unsolved**: Not already fixed by frameworks?
5. ✅ **Bug vs Feature**: Actual pain vs nice-to-have?
6. ✅ **Scope**: Widespread or niche use case?

**Overall Results**: 39% validation rate (9 out of 23 skills)

**Tier Breakdown**:
- Production (Batch 1 - Infrastructure): **83% validation** ✅
- Beta (Batch 2-4): **0-50% validation** 🟡
- Experimental: **0% validation** ⚠️

## Tier Philosophy

### Why Three Tiers?

**Transparency**: We don't hide market realities
**User Choice**: Advanced users may have specific needs
**Usage Data**: Let the market validate Beta skills
**Promotion Path**: High-usage Beta → Production based on metrics

### Promotion Criteria (Beta → Production)
- 100+ installs
- 4.0+ average rating
- <5% uninstall rate
- Positive user feedback
